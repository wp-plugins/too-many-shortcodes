All of your shortcodes on one settings page.

== Description ==

View or change the trigger text, enable or disable individual shortcodes, see the optional arguments and explaination links for each shortcode, all on one page. New shortcodes added with every update.


== Installation ==
1. Download the latest zip file and extract the `too-many-shortcodes` directory.
2. Upload this directory inside your `/wp-content/plugins/` directory.
3. Activate 'Too Many Shortcodes' on the 'Plugins' menu in WordPress.
